Requirements:
```
Python 3.7.1 or newer
Virtualenv
Postgres installed locally with PATH set up for psql
```

Running locally:
```
$ cd learning-flask
$ virtualenv flaskenv
$ pip3 install -r requirements.txt
$ source flaskenv/bin/activate #On Mac
$ . .\flaskenv\Scripts\activate.ps1 #On Windows
$ python3 routes.py
```
